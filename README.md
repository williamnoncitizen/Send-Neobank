# Send · Path to Neobank

Internal product strategy dashboard for Send's transition from social money app to neobank.

## Pages

- [`index.html`](./index.html) — **Path to Neobank**: maturity ladder, persona, chasm framework, prioritization matrix, ship-next stack.
- [`matrix.html`](./matrix.html) — **Comp Analysis**: 25 features × 12 competitors interactive matrix.

## Stack

Pure static HTML, CSS, and vanilla JavaScript. No build step. Hosted on GitHub Pages.
